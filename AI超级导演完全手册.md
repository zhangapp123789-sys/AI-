## 一、导演思维：AI漫剧的本质

| 传统思维 | AI导演思维 |
|----------|-----------|
| "我要拍一部剧" | "我要把一段文字变成会动的画面" |
| 先招团队，再开拍 | 先定流程，再调AI |
| 拍了再看效果 | 先出样片，再迭代 |

**核心公式：**
```
好故事 × 统一角色 × 电影级画面 × 情绪化配音 ÷ 制作成本 = AI漫剧竞争力
```

## 三、标准化60秒模板

```
0-3秒    │ 强钩子 ─ 特写+冲突+悬念                    │ 抓住眼球
3-12秒   │ 交代背景 ─ 中景+环境+人物状态              │ 建立认知
12-35秒  │ 核心剧情 ─ 近景+情绪变化+动作高潮          │ 讲清故事
35-55秒  │ 反转/爽点 ─ 特效+节奏提速                   │ 情绪爆发
55-60秒  │ 悬念 ─ 黑屏+字幕「下一集更精彩」            │ 追更引导
```
