## 二、角色一致性四步法

### 第1步：生成角色定妆照
```
即梦AI → 输入角色卡描述 → 生成3-5张 → 选最满意的一张作为「参考图」
```

### 第2步：每场景引用参考图
```
即梦AI → 上传参考图 → 提示词开头加「@图片1 同一角色」→ 画面变了衣服和场景，脸不变
```

### 第3步：小云雀/HappyHorse继续引用
```
小云雀 → 导入参考图 → 描述场景和动作 → 角色保持一致
HappyHorse I2V → image_url传入角色图 → 所有场景同一张脸
```

### 第4步：跨集复用
```
第二集不重新生成角色 → 直接用第一集的参考图 → 换场景提示词即可
```

## 五、口型同步方案

### 方案A：先配音后对嘴型（推荐）

```
第1步：edge-tts生成配音 → 得到准确时长和语调
第2步：把视频+配音一起导入小云雀 → 自动对口型
第3步：或使用HeyGen API → 上传视频+音频 → 自动Lip Sync
```

### 方案B：HappyHorse原生音画同步

```
HappyHorse自带音视频联合生成能力——
提示词中加对话内容，模型在生成视频的同时生成同步音轨。
但目前T2V模式不支持旁白配音，只支持角色对话。
```

### 口型提示词（英文，给AI视频工具用）：

```
# 说话中
man speaking Chinese narrating, mouth opening and closing naturally,
lips forming words, slight jaw movement, normal speaking pace

# 惊讶
man gasping, mouth wide open, lips parting slowly, 
eyes widening simultaneously with mouth movement

# 紧张
man breathing heavily, lips pressed tight, jaw clenching,
occasional lip licking, throat swallowing visibly

# 绝望
mouth dropping open in silent scream, lips trembling,
jaw shaking, mouthing "impossible" repeatedly
```
